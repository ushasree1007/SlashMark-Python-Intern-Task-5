## APSCHE Python Internship – Task 5 Completed

Successfully completed **Task 5** of the APSCHE Python internship program.

### Project: Personal Voice Assistant

Built a Python-based personal assistant application that helps users access different platforms and basic utilities through a simple menu-driven system. The assistant provides voice responses and performs tasks based on user input.

### Features

* Voice interaction using text-to-speech
* Menu options for quick access
* Opens websites through numeric selection
* Shows current date and time
* Simple and interactive user interface
* Python automation concepts implemented

### Tools Used

* Python
* pyttsx3
* webbrowser
* datetime

### Outcome

This task helped me improve practical skills in Python automation, basic assistant development, and integrating external libraries for real-time interaction.
